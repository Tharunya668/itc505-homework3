Welcome to My Font-Style Website

    Explanation of Font Delivery Process:
  1. This is a paragraph of text with the "kablammo" font style
  2. In this assignment, I'm delivering a custom font to the client's browser by using Google Fonts.
  3. I visited the Google Fonts website (https://fonts.google.com/).
  4. I selected the "kablammo" font, which is not typically present on most computers.
  5. I obtained a link to the font in the "Embed Font" section of Google Fonts.
  6. I linked to the font in my HTML by adding the following link tag: <link rel="stylesheet" href="https://fonts.googleapis.com/css2?family=kablammo&display=swap">
  7. In my CSS file (style.css), I applied the "kablammo" font to various elements.
  8. The font does not look conventional, but it's readable and noticeable.
